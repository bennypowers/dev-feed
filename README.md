# \<dev-feed>

This webcomponent follows the [open-wc](https://github.com/open-wc/open-wc) recommendation.

## Installation
```bash
npm i dev-feed
```

## Usage
```html
<script type="module">
  import 'dev-feed/dev-feed.js';
</script>

<dev-feed></dev-feed>
```
